# 実装計画 (Implementation Plan)

## 1. 目的
`docs/readmes` に、利用者向けの運用手順と設計者向けの仕様整理を追加する。

## 2. 作成ドキュメント
1. 使い方ガイド
   - セットアップ
   - 起動確認
   - アプリ追加手順
   - DNS 5353 暫定運用と 53 本番運用
2. アーキテクチャ仕様書
   - 設計原則
   - レイヤー構成
   - データ/ネットワークの流れ
   - 制約と拡張余地

## 3. 確認観点
- 既存実装とドキュメント内容が矛盾しない
- 初見利用者が `setup.sh` 実行から UI 操作まで辿れる
- 53 番競合対策（`DNS_HOST_PORT=5353`）が明示されている
