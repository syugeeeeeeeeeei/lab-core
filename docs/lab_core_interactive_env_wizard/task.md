# Task List: 対話型 .env 設定ウィザード

## 目的
`.env` を直接手編集しなくても安全に設定できるようにし、設定ミスへの不安を減らす。

## 実施タスク
- `yarn config:init` を追加
- `yarn config:reset` を追加
- 各設定項目に用途/役割/設定例を表示する対話フローを実装
- 既存 `.env` 上書き時の自動バックアップを実装
- backend 起動時の `.env` 自動読込を実装
- README / 説明書へ手順を追記

## 完了条件
- 利用者が `.env` を直接編集せずに設定できる
- 保存前に値確認ができる
- `.env` が backend 起動時に反映される
