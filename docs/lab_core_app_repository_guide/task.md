# Task List: 適合アプリリポジトリ作成ガイド

## 目的
Lab-Core v3 に載せるアプリリポジトリを、開発者が迷わず作成できるようにする。

## 実施タスク
- 現行実装における受け入れ条件を整理
- compose/Dockerfile/ログ/永続化の作法を文書化
- 登録値テンプレートと自己チェック手順を追加
- よくある失敗パターンと受け入れ判定チェックリストを追加
- OruCa 系（NFC/Slack）向けの補足を追加

## 完了条件
- 新規アプリ開発者がこの文書だけで登録準備を完了できる
- `publicServiceName/publicPort/composePath` の不整合を事前に防げる
- 運用側が受け入れ判定を定型化できる
