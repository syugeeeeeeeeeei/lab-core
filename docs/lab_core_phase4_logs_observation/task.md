# Task List: Phase 4 ログ監視導線の実装

## 目的
障害時に「再起動の次にログを見る」導線を UI 上で完結させる。

## 実施タスク
- バックエンドにアプリログ取得 API を追加
- バックエンドにログ対象サービス一覧 API を追加
- Dashboard のアプリ一覧に「ログ」ボタンを追加
- ログビューア（サービス切替、表示行数切替、自動スクロール）を追加
- エラー/警告を視覚的に判別しやすくする
- 説明書とユーザーテスト手順を更新する

## 完了条件
- UI からアプリ単位でログを閲覧できる
- サービス切替・表示行数変更で再取得できる
- `dry-run` と `execute` で挙動の違いが説明書に記載されている
